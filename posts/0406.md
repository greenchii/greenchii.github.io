## 283-移动0

给定一个数组 `nums`，编写一个函数将所有 `0` 移动到数组的末尾，同时保持非零元素的相对顺序。

**请注意** ，必须在不复制数组的情况下原地对数组进行操作。

**示例 1:**

```plain&#x20;text
输入: nums = [0,1,0,3,12]输出: [1,3,12,0,0]
```

**示例 2:**

```plain&#x20;text
输入: nums = [0]输出: [0]
```

**<span style="color: rgb(216,57,49); background-color: inherit">双指针</span> <span style="color: rgb(216,57,49); background-color: inherit">的想法：</span>**

<span style="color: rgb(216,57,49); background-color: inherit">准备两个指针，都从index 0出发：</span>

* `fast`<span style="color: rgb(216,57,49); background-color: inherit">：负责</span>**<span style="color: rgb(216,57,49); background-color: inherit">扫描</span>**<span style="color: rgb(216,57,49); background-color: inherit">，从头跑到尾，每个元素都看</span>

* `slow`<span style="color: rgb(216,57,49); background-color: inherit">：负责</span>**<span style="color: rgb(216,57,49); background-color: inherit">占位</span>**<span style="color: rgb(216,57,49); background-color: inherit">，记录&quot;下一个非零元素应该填到哪里&quot;</span>

<span style="color: rgb(216,57,49); background-color: inherit">规则只有一条：</span>**<span style="color: rgb(216,57,49); background-color: inherit">fast遇到非零元素，就填到slow的位置，然后slow往右移一格</span>**

```c++
初始：
[0, 1, 0, 3, 12]
 ↑
slow=0, fast=0

fast看到0，是零，不管，fast右移：
[0, 1, 0, 3, 12]
 ↑  ↑
slow fast

fast看到1，非零！把1填到slow(0)的位置，slow右移：
[1, 1, 0, 3, 12]
    ↑  ↑
   slow fast

fast看到0，是零，不管，fast右移：
[1, 1, 0, 3, 12]
    ↑     ↑
   slow  fast

fast看到3，非零！把3填到slow(1)的位置，slow右移：
[1, 3, 0, 3, 12]
       ↑     ↑
      slow  fast

fast看到12，非零！把12填到slow(2)的位置，slow右移：
[1, 3, 12, 3, 12]
          ↑
         slow=3，fast结束

最后slow=3之后全填0：
[1, 3, 12, 0, 0]  
```

```c++
class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        int fast=0,slow=0;
        int n=nums.size();
        //fast遇到非零数就把数填到slow的位置上然后slow向右移
        while(fast<n){
            if(nums[fast]!=0){
                nums[slow]=nums[fast];
                slow++;
            }
            fast++;
        }

        //slow及之后的位置填0
        while(slow<n){
            nums[slow]=0;
            slow++;
        }
    }
};
```

## 11-盛最多水的容器

找出其中的两条线，使得它们与 `x` 轴共同构成的容器可以容纳最多的水。

### 暴力解法

```python
class Solution {
public:
    int maxArea(vector<int>& height) {
        int n=height.size();
        int len,wid,res=0;
        for(int i=0;i<n;i++){
            for(int j=i+1;j<n;j++){
                len=j-i;
                wid=min(height[i],height[j]);
                res=max(res,len*wid);
            }
        }
        return res;
    }
};
```

## 双指针

**<span style="color: rgb(216,57,49); background-color: inherit">容器面积公式：</span>**

```plain&#x20;text
面积 = 宽度 × 高度
     = (right - left) × min(height[left], height[right])
```

<span style="color: rgb(216,57,49); background-color: inherit">两个指针从最两端出发，此时</span>**<span style="color: rgb(216,57,49); background-color: inherit">宽度最大</span>**<span style="color: rgb(216,57,49); background-color: inherit">。</span>

***

**<span style="color: rgb(216,57,49); background-color: inherit">每一步怎么移动？</span>**

<span style="color: rgb(216,57,49); background-color: inherit">假设当前状态：</span>

```plain&#x20;text
[1, 8, 6, 2, 5, 4, 8, 3, 7]
 ↑                          ↑
left=0(高1)            right=8(高7)
```

<span style="color: rgb(216,57,49); background-color: inherit">面积 = 8 × min(1, 7) = </span>**<span style="color: rgb(216,57,49); background-color: inherit">8×1 = 8</span>**

<span style="color: rgb(216,57,49); background-color: inherit">现在考虑：移左还是移右？</span>

* <span style="color: rgb(216,57,49); background-color: inherit">当前短板是左边（高1）</span>

* <span style="color: rgb(216,57,49); background-color: inherit">如果移右边：宽度变小，高度还是受限于左边的1，面积</span>**<span style="color: rgb(216,57,49); background-color: inherit">只会更小</span>**

* <span style="color: rgb(216,57,49); background-color: inherit">如果移左边：宽度变小，但高度</span>**<span style="color: rgb(216,57,49); background-color: inherit">有可能变大</span>**<span style="color: rgb(216,57,49); background-color: inherit">，面积有机会更大</span>

<span style="color: rgb(216,57,49); background-color: inherit">所以</span>**<span style="color: rgb(216,57,49); background-color: inherit">移较矮的那边</span>**<span style="color: rgb(216,57,49); background-color: inherit">，放弃较高的那边。</span>

```sql
class Solution {
public:
    int maxArea(vector<int>& height) {
        int n=height.size();
        int left=0,right=n-1;
        int res=0,area=0;
        while(left<right){
            area=(right-left)*min(height[right],height[left]);
            res=max(area,res);
            if(height[left]<height[right])left++;
            else right--;
        }
        return res;
    }
};
```

## 15-三数之和

给你一个整数数组 `nums` ，判断是否存在三元组 `[nums[i], nums[j], nums[k]]` 满足 `i != j`、`i != k` 且 `j != k` ，同时还满足 `nums[i] + nums[j] + nums[k] == 0` 。请你返回所有和为 `0` 且不重复的三元组。

**注意：**&#x7B54;案中不可以包含重复的三元组。

**示例 1：**

**输入：**&#x6E;ums = \[-1,0,1,2,-1,-4]
**输出：**\[\[-1,-1,2],\[-1,0,1]]
**解释：**
nums\[0] + nums\[1] + nums\[2] = (-1) + 0 + 1 = 0 。
nums\[1] + nums\[2] + nums\[4] = 0 + 1 + (-1) = 0 。
nums\[0] + nums\[3] + nums\[4] = (-1) + 2 + (-1) = 0 。
不同的三元组是 \[-1,0,1] 和 \[-1,-1,2] 。
注意，输出的顺序和三元组的顺序并不重要。

**示例 2：**

**输入：**&#x6E;ums = \[0,1,1]
**输出：**\[]
**解释：**&#x552F;一可能的三元组和不为 0 。

**示例 3：**

**输入：**&#x6E;ums = \[0,0,0]
**输出：**\[\[0,0,0]]
**解释：**&#x552F;一可能的三元组和为 0 。

### 数组去重方法

1. **用 `unordered_set`（最常用）**

```c++
vector<int> nums = {1, 2, 2, 3, 3, 3, 4};
unordered_set<int> s(nums.begin(), nums.end());
vector<int> res(s.begin(), s.end());
// res = {1, 2, 3, 4}（顺序不保证）
```

***

* **先排序，再 `unique`（保持有序）**

```c++
vector<int> nums = {1, 2, 2, 3, 3, 3, 4};
sort(nums.begin(), nums.end());
nums.erase(unique(nums.begin(), nums.end()), nums.end());
// nums = {1, 2, 3, 4}
```

`unique` 会把重复元素移到末尾，返回新的末尾迭代器，`erase` 把它们删掉。

1. **双指针原地去重（适合已排序数组，不开额外空间）**

```c++
// 和移动零那道题思路一样
vector<int> nums = {1, 1, 2, 3, 3, 4};  // 必须已排序
int slow = 0;
for (int fast = 1; fast < nums.size(); fast++) {
    if (nums[fast] != nums[slow]) {  // 遇到新元素
        slow++;
        nums[slow] = nums[fast];
    }
}
nums.resize(slow + 1);
// nums = {1, 2, 3, 4}
```

<span style="color: rgb(216,57,49); background-color: inherit">三个数，乍一看是三指针，但是如果我们将问题转化为求两个数的和等于sum-另一个数，那就转化为了我们熟悉的</span> <span style="color: rgb(216,57,49); background-color: inherit">双指针</span>

<span style="color: rgb(216,57,49); background-color: inherit">因此思路是固定一个数（for外循环固定，内部使用</span> <span style="color: rgb(216,57,49); background-color: inherit">双指针</span> <span style="color: rgb(216,57,49); background-color: inherit">）</span>

```c++
vector<vector<int>> threeSum(vector<int>& nums) {
    sort(nums.begin(), nums.end());       // 第一步：排序
    vector<vector<int>> res;
    int n = nums.size();

    for (int i = 0; i < n - 2; i++) {    // 第二步：固定i
    //i 最多到 n-3，因为 left 和 right 至少还要占两个位置。
        if (i > 0 && nums[i] == nums[i-1]) continue;  // i去重 防止固定同一个数；i>0是为了防止nums[i-1]越界
        if (nums[i] > 0) break;                         // 剪枝
        //数组已排序，如果 nums[i] > 0，那 left 和 right 都比 nums[i] 大，三数之和必然 > 0，不可能等于0，直接结束。

        int left = i + 1, right = n - 1; // 第三步：双指针
        while (left < right) {
            int sum = nums[i] + nums[left] + nums[right];
            if (sum == 0) {
                res.push_back({nums[i], nums[left], nums[right]});
                while (left < right && nums[left] == nums[left+1]) left++;//left去重
                while (left < right && nums[right] == nums[right-1]) right--;//right去重
                left++;
                right--;
            } else if (sum < 0) {
                left++;
            } else {
                right--;
            }
        }
    }
    return res;
}
```

## 42-接雨水

给定 `n` 个非负整数表示每个宽度为 `1` 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

**示例 1：**

![](https://my.feishu.cn/space/api/box/stream/download/asynccode/?code=MWVjOWE3ZGY0YzM4YWJiZjg1NGY4ZTIxMTBmMDVmNTFfMlJURjFrdmd5emtrMUx5MHdwY0NNamNIaEZYa3N3NG5fVG9rZW46VlZLNGJJaklxb1dmaXB4dlY0R2N4ZnRIbmFnXzE3NzU0NzU2NzI6MTc3NTQ3OTI3Ml9WNA)

**输入：**&#x68;eight = \[0,1,0,2,1,0,1,3,2,1,2,1]
**输出：**&#x36;
**解释：**&#x4E0A;面是由数组 \[0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图，在这种情况下，可以接 6 个单位的雨水（蓝色部分表示雨水）。&#x20;

**示例 2：**

**输入：**&#x68;eight = \[4,2,0,3,2,5]
**输出：**&#x39;

<span style="color: rgb(216,57,49); background-color: inherit">对于每个位置 </span>`i`<span style="color: rgb(216,57,49); background-color: inherit">，它能接的水量取决于：</span>

* <span style="color: rgb(216,57,49); background-color: inherit">它左边最高的柱子 </span>`leftMax`

* <span style="color: rgb(216,57,49); background-color: inherit">它右边最高的柱子 </span>`rightMax`

<span style="color: rgb(216,57,49); background-color: inherit">水位由</span>**<span style="color: rgb(216,57,49); background-color: inherit">两边较矮的那个</span>**<span style="color: rgb(216,57,49); background-color: inherit">决定，减去当前柱子高度就是这个位置接的水：</span>

```plain&#x20;text
water[i] = min(leftMax, rightMax) - height[i]
```

**<span style="color: rgb(216,57,49); background-color: inherit">双指针</span> <span style="color: rgb(216,57,49); background-color: inherit">的核心逻辑：</span>**

`leftMax < rightMax`<span style="color: rgb(216,57,49); background-color: inherit"> 时，处理左边：</span>

* <span style="color: rgb(216,57,49); background-color: inherit">左边的水位由 </span>`leftMax`<span style="color: rgb(216,57,49); background-color: inherit"> 决定（因为右边一定有更高的墙挡着）</span>

* <span style="color: rgb(216,57,49); background-color: inherit">当前位置接水 = </span>`leftMax - height[left]`

* <span style="color: rgb(216,57,49); background-color: inherit">left右移</span>

<span style="color: rgb(216,57,49); background-color: inherit">反之处理右边，right左移。</span>

```sql
int trap(vector<int>& height) {
    int left = 0, right = height.size() - 1;
    int leftMax = 0, rightMax = 0;
    int res = 0;

    while (left < right) {
        leftMax = max(leftMax, height[left]);
        rightMax = max(rightMax, height[right]);

        if (leftMax < rightMax) {
            res += leftMax - height[left];
            left++;
        } else {
            res += rightMax - height[right];
            right--;
        }
    }
    return res;
}
```
