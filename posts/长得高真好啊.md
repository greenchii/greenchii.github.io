# 3-无重复字符的最长子串

给定一个字符串 `s` ，请你找出其中不含有重复字符的 **最长&#x20;**&#x5B50;&#x4E32;**&#x20;**&#x7684;长度。

**示例 1:**

```plain&#x20;text
输入: s = "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。注意 "bca" 和 "cab" 也是正确答案。
```

**示例 2:**

```plain&#x20;text
输入: s = "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。
```

**示例 3:**

```plain&#x20;text
输入: s = "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。
```

### <span style="color: rgb(220,155,4); background-color: inherit">本题解法</span>

<span style="color: rgb(220,155,4); background-color: inherit">我们用两个指针 </span>`left`<span style="color: rgb(220,155,4); background-color: inherit"> 和 </span>`right`<span style="color: rgb(220,155,4); background-color: inherit"> 表示当前考察的无重复子串 </span>`s[left..right]`<span style="color: rgb(220,155,4); background-color: inherit">。</span>

**<span style="color: rgb(220,155,4); background-color: inherit">需要解决的核心问题：</span>**<span style="color: rgb(220,155,4); background-color: inherit"> </span>`right`<span style="color: rgb(220,155,4); background-color: inherit"> 右移一步，遇到字符 </span>`s[right]`<span style="color: rgb(220,155,4); background-color: inherit"> 时，如何快速判断它是否已经在窗口内出现过？</span>

**<span style="color: rgb(220,155,4); background-color: inherit">解决方案：</span>**<span style="color: rgb(220,155,4); background-color: inherit"> 维护一张</span> <span style="color: rgb(220,155,4); background-color: inherit">哈希表</span> <span style="color: rgb(220,155,4); background-color: inherit"> </span>`last`<span style="color: rgb(220,155,4); background-color: inherit">，</span>**<span style="color: rgb(220,155,4); background-color: inherit">键是字符，值是该字符最近一次出现的下标</span>**<span style="color: rgb(220,155,4); background-color: inherit">。这样当 </span>`right`<span style="color: rgb(220,155,4); background-color: inherit"> 扫到 </span>`s[right]`<span style="color: rgb(220,155,4); background-color: inherit"> 时：</span>

* <span style="color: rgb(220,155,4); background-color: inherit">若 </span>`last`<span style="color: rgb(220,155,4); background-color: inherit"> 中记录的 </span>`s[right]`<span style="color: rgb(220,155,4); background-color: inherit"> 的上次位置 ≥\geq ≥</span>`left`<span style="color: rgb(220,155,4); background-color: inherit">，说明它还在窗口内，出现了重复 → 把 </span>`left`<span style="color: rgb(220,155,4); background-color: inherit"> 跳到那个位置 +1+1 +1，将重复字符踢出窗口 </span>

* <span style="color: rgb(220,155,4); background-color: inherit">否则，</span>`s[right]`<span style="color: rgb(220,155,4); background-color: inherit"> 不在窗口内，直接纳入，无需收缩</span>

<span style="color: rgb(220,155,4); background-color: inherit">每步更新 </span>`last[s[right]] = right`<span style="color: rgb(220,155,4); background-color: inherit">，并用 </span>`right - left + 1`<span style="color: rgb(220,155,4); background-color: inherit"> 刷新答案。</span>

**<span style="color: rgb(220,155,4); background-color: inherit">走一遍示例 </span>`"abcabcbb"`<span style="color: rgb(220,155,4); background-color: inherit">：</span>**

![](https://my.feishu.cn/space/api/box/stream/download/asynccode/?code=MWM4M2FhZWRhZTZkMzdkZTJlYzNjNjZhODc5YjllYzJfR0tScExRV3oyODN1alpjWUpwR1JzalowZkYwZTZTb0lfVG9rZW46UUlKZGJubVo2b1RGUHV4dkRSTWNpT3FmbnlmXzE3NzYzNDU0NjI6MTc3NjM0OTA2Ml9WNA)

```c++
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        unordered_map<char,int> map;//用于存放已遍历字符与下标
        int left=0;
        int ans=0;
        for(int right=0;right<s.length();right++){
            char cur=s[right];//当前字符

            if(map.count(cur)&&map[cur]>=left){//如果遍历过并且在left右边或是left，说明在窗口中，该字符重复
                left=map[cur]+1;//从上一个重复字符后重新开始计算窗口
            }
            map[cur]=right;
            ans=max(ans,right-left+1);//每遇一个字符就要更新计数
        }
        return ans;
    }
};
```

# 438-找到字符串中所有字母异位词

<span style="color: rgb(143,149,158); background-color: inherit">给定两个字符串 </span>`s`<span style="color: rgb(143,149,158); background-color: inherit"> 和 </span>`p`<span style="color: rgb(143,149,158); background-color: inherit">，找到 </span>`s`**<span style="color: rgb(143,149,158); background-color: inherit"> </span>**<span style="color: rgb(143,149,158); background-color: inherit">中所有 </span>`p`**<span style="color: rgb(143,149,158); background-color: inherit"> </span>**<span style="color: rgb(143,149,158); background-color: inherit">的 异位词</span>**<span style="color: rgb(143,149,158); background-color: inherit"> </span>**<span style="color: rgb(143,149,158); background-color: inherit">的子串，返回这些子串的起始索引。不考虑答案输出的顺序。</span>

**<span style="color: rgb(143,149,158); background-color: inherit">示例 1:</span>**

**<span style="color: rgb(143,149,158); background-color: inherit">输入: </span>**<span style="color: rgb(143,149,158); background-color: inherit">s = &quot;cbaebabacd&quot;, p = &quot;abc&quot;</span>
**<span style="color: rgb(143,149,158); background-color: inherit">输出: </span>**<span style="color: rgb(143,149,158); background-color: inherit">[0,6]</span>
**<span style="color: rgb(143,149,158); background-color: inherit">解释:</span>** <span style="color: rgb(143,149,158); background-color: inherit">起始索引等于 0 的子串是 &quot;cba&quot;, 它是 &quot;abc&quot; 的异位词。</span> <span style="color: rgb(143,149,158); background-color: inherit">起始索引等于 6 的子串是 &quot;bac&quot;, 它是 &quot;abc&quot; 的异位词。</span>

**<span style="color: rgb(143,149,158); background-color: inherit"> 示例 2:</span>**

**<span style="color: rgb(143,149,158); background-color: inherit">输入: </span>**<span style="color: rgb(143,149,158); background-color: inherit">s = &quot;abab&quot;, p = &quot;ab&quot;</span>
**<span style="color: rgb(143,149,158); background-color: inherit">输出: </span>**<span style="color: rgb(143,149,158); background-color: inherit">[0,1,2]</span>
**<span style="color: rgb(143,149,158); background-color: inherit">解释:</span>** <span style="color: rgb(143,149,158); background-color: inherit">起始索引等于 0 的子串是 &quot;ab&quot;, 它是 &quot;ab&quot; 的异位词。</span> <span style="color: rgb(143,149,158); background-color: inherit">起始索引等于 1 的子串是 &quot;ba&quot;, 它是 &quot;ab&quot; 的异位词。</span> <span style="color: rgb(143,149,158); background-color: inherit">起始索引等于 2 的子串是 &quot;ab&quot;, 它是 &quot;ab&quot; 的异位词。</span>

### 什么是异位词？

异位词就是字符种类和每种字符的数量完全相同，只是顺序不同的字符串。比如 `"abc"` 和 `"cba"` 是异位词。

判断两个字符串是否互为异位词，最直接的方法就是**比较两者的字符频次**是否完全一致。

***

### 为什么用滑动窗口？

题目要找 `s` 中所有长度等于 `p.length()` 的子串，判断它是否是 `p` 的异位词。

暴力做法是枚举 `s` 中每个长度为 `p.length()` 的子串，各自统计频次再比较，时间复杂度 O(n⋅∣Σ∣)O(n \cdot |\Sigma|) O(n⋅∣Σ∣)。&#x20;

但注意：相邻两个窗口只差一个字符（右边进一个，左边出一个），没必要每次从头统计。用**固定长度的滑动窗口**，每次只更新进出的那两个字符的频次，就能把比较的开销分摊掉。

### 本题解法

**准备两个频次数组：**

* `pCount[26]`：统计 `p` 中每个字母的出现次数，初始化一次即可，之后不变

* `sCount[26]`：统计当前窗口 `s[left..right]` 中每个字母的出现次数

窗口长度固定为 `p.length()`，设为 `k`。

**滑动过程：**

* `right` 从 0 开始右移，将 `s[right]` 加入窗口，`sCount[s[right]]++`

* 当窗口长度达到 `k`（即 `right >= k-1`）时，比较 `sCount == pCount`，相等则记录 `left`

* 然后将 `s[left]` 移出窗口，`sCount[s[left]]--`，`left++`

![](https://my.feishu.cn/space/api/box/stream/download/asynccode/?code=ZWE0MmFkNzMyM2M5NjUxZjc2Y2ZiZjRmZmY2MWE3MTJfWW9mNWVVUE1XaDZGY3VuTmh1V1E0NDZMdlltRFc2UVNfVG9rZW46VzlFdWJoTjFOb3AxNFp4a3lDcmM4SFJ6blNlXzE3NzYzNDU0NjI6MTc3NjM0OTA2Ml9WNA)

```c++
class Solution {
public:
    vector<int> findAnagrams(string s, string p) {
        vector<int> ans;

        // s 比 p 短，不可能存在异位词
        if (s.length() < p.length()) return ans;

        int k = p.length(); // 窗口固定长度

        // pCount：p 的字符频次（固定不变）
        // sCount：当前窗口的字符频次（随窗口滑动更新）
        vector<int> pCount(26, 0), sCount(26, 0);

        // 初始化 pCount
        for (char c : p) pCount[c - 'a']++;

        int left = 0;
        for (int right = 0; right < s.length(); right++) {
            // s[right] 进入窗口，更新频次
            sCount[s[right] - 'a']++;

            // 窗口长度达到 k，开始判断
            if (right - left + 1 == k) {
                // 当前窗口字符频次与 p 完全一致，说明是异位词
                if (sCount == pCount) {
                    ans.push_back(left);
                }
                // s[left] 移出窗口，更新频次，左边界右移
                sCount[s[left] - 'a']--;
                left++;
            }
        }

        return ans;
    }
};
```

# 53-最大子数组和

给你一个整数数组 `nums` ，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

子数组是数组中的一个连续部分。

**示例 1：**

**输入：**&#x6E;ums = \[-2,1,-3,4,-1,2,1,-5,4]
**输出：**&#x36;
**解释：**&#x8FDE;续子数组 \[4,-1,2,1] 的和最大，为 6 。

**示例 2：**

**输入：**&#x6E;ums = \[1]
**输出：**&#x31;

**示例 3：**

**输入：**&#x6E;ums = \[5,4,-1,7,8]
**输出：**&#x32;3

```c++
class Solution {
public:
    int maxSubArray(vector<int>& nums) {
         int cur = nums[0];   // 以当前位置结尾的最大子数组和
        int ans = nums[0];   // 全局最大值

        for (int i = 1; i < nums.size(); i++) {
            cur = max(nums[i], cur + nums[i]);
            ans = max(ans, cur);
        }

        return ans;
    }
};
```

## 分治思路

将数组从中间切开，最大子数组只有三种可能：

```plain&#x20;text
[        left        |        right       ]
 ①完全在左半部分      ②完全在右半部分      ③跨越中点
```

① ② 递归解决，③ 需要单独计算。

***

## 关键：如何求跨越中点的最大子数组？

跨越中点的子数组**必须包含 mid 和 mid+1**，所以：

* 从 `mid` 向左延伸，找最大前缀和

* 从 `mid+1` 向右延伸，找最大后缀和

* 两者相加即为跨越中点的最大值

```plain&#x20;text
... [ ? ? ? | mid ] [ mid+1 | ? ? ? ] ...
        ←左延伸         右延伸→
```



```c++
// 计算跨越中点的最大子数组和
int crossMax(vector<int>& nums, int left, int mid, int right) {
    // 从 mid 向左，找最大左半和
    int leftSum = INT_MIN, cur = 0;
    for (int i = mid; i >= left; i--) {
        cur += nums[i];
        leftSum = max(leftSum, cur);
    }

    // 从 mid+1 向右，找最大右半和
    int rightSum = INT_MIN; cur = 0;
    for (int i = mid + 1; i <= right; i++) {
        cur += nums[i];
        rightSum = max(rightSum, cur);
    }

    return leftSum + rightSum;
}

int solve(vector<int>& nums, int left, int right) {
    // base case：只有一个元素
    if (left == right) return nums[left];

    int mid = left + (right - left) / 2;

    int leftAns  = solve(nums, left, mid);       // ① 左半部分
    int rightAns = solve(nums, mid + 1, right);  // ② 右半部分
    int crossAns = crossMax(nums, left, mid, right); // ③ 跨越中点

    return max({leftAns, rightAns, crossAns});
}

int maxSubArray(vector<int>& nums) {
    return solve(nums, 0, nums.size() - 1);
}
```

# 560-和为k的数组

## 算法讲解

### 核心知识点：前缀和 + 哈希表

**前缀和定义：**

* 设 `prefix[i]` = `nums[0] + nums[1] + ... + nums[i-1]`

* 则子数组 `nums[l..r]` 的和 = `prefix[r+1] - prefix[l]`

**问题转化：**

我们要找满足 `sum(nums[l..r]) == k` 的子数组个数，即：

$$\text{prefix}[r+1] - \text{prefix}[l] = k$$

变形得：

$$\text{prefix}[l] = \text{prefix}[r+1] - k$$

所以，**每遍历到位置 `r`，只需查询之前出现过多少次值为 `(当前前缀和 - k)` 的前缀和**，这就是新增的合法子数组数。

### 为什么不能用滑动窗口？

滑动窗口要求单调性（如全为正数）。本题 `nums` 含负数，窗口扩大时和不一定增大，无法维护双指针，**必须用前缀和 + 哈希表**。

### 算法流程

```plain&#x20;text
初始化：哈希表 map = {0: 1}（空前缀，和为0出现1次）
        当前前缀和 cur = 0，答案 ans = 0

遍历每个元素 x：
    cur += x
    ans += map[cur - k]    // 查找之前有多少前缀和等于 cur-k
    map[cur]++             // 记录当前前缀和
```

### 图解示例

`nums = [1, 1, 1]`，`k = 2`

答案为 **2** ✓

### 复杂度

* **时间**：$O(n)$，一次遍历

* **空间**：$O(n)$，哈希表最多存 $n+1$ 个前缀和

***

## C++ 代码

```c++
#include <unordered_map>
#include <vector>
using namespace std;

class Solution {
public:
    int subarraySum(vector<int>& nums, int k) {
        // 哈希表：记录各前缀和出现的次数
        unordered_map<int, int> prefixCount;//下标表示前缀和是多少，值表示出现过几次
        
        // 初始化：前缀和为 0 的情况出现 1 次（空前缀）
        // 处理子数组从 nums[0] 开始的情况
        prefixCount[0] = 1;
        
        int cur = 0;  // 当前前缀和
        int ans = 0;  // 满足条件的子数组个数
        
        for (int x : nums) {
            cur += x;  // 更新前缀和
            
            // 查询之前是否存在前缀和等于 (cur - k)
            // 若存在，说明从那之后到当前位置的子数组和恰好为 k
            if (prefixCount.count(cur - k)) {
                ans += prefixCount[cur - k];
            }
            
            // 将当前前缀和计入哈希表
            prefixCount[cur]++;
        }
        
        return ans;
    }
};
```

***

## 关键细节回顾

**为什么初始化 `prefixCount[0] = 1`？**

如果子数组从下标 0 开始，比如 `nums[0..r]` 的和恰好等于 `k`，则：

```plain&#x20;text
cur - k == 0
```

此时需要在 map 中找到 `0`，所以必须预先放入 `{0: 1}`，否则会漏计这类情况。

# 279-完全平方数

## 算法讲解

### 问题分析

本质是一个**完全背包 + 最小值**问题：

* "物品"：所有不超过 n 的完全平方数 `1, 4, 9, 16, ...`

* "目标"：用最少数量的物品，使得总和恰好等于 n

* 每个完全平方数可以**无限次使用**（完全背包）

### DP 定义

$$dp[i] = \text{凑成整数 } i \text{ 所需完全平方数的最少个数}$$

**初始化：**

* `dp[0] = 0`（凑成 0 不需要任何数）

* `dp[1..n] = INF`（初始不可达）

**状态转移：**

对于每个 `i`，枚举所有满足 `j*j <= i` 的完全平方数 `j*j`：

$$dp[i] = \min(dp[i],\ dp[i - j \cdot j] + 1)$$

含义：从 `dp[i - j*j]` 的最优解，再加上一个 `j*j`，得到 `dp[i]`。

***

### 图解过程（n = 12）

完全平方数候选：1, 4, 9

```plain&#x20;text
dp[0] = 0
dp[1] = dp[0] + 1 = 1        (1)
dp[2] = dp[1] + 1 = 2        (1+1)
dp[3] = dp[2] + 1 = 3        (1+1+1)
dp[4] = min(dp[3]+1, dp[0]+1) = 1    (4)
dp[5] = min(dp[4]+1, dp[1]+1) = 2    (4+1)
...
dp[9] = min(..., dp[0]+1) = 1        (9)
dp[10]= min(dp[9]+1, dp[6]+1) = 2    (9+1)
dp[11]= min(dp[10]+1,...) = 3        (9+1+1)
dp[12]= min(dp[11]+1, dp[8]+1, dp[3]+1) = 3  (4+4+4)
```

答案 `dp[12] = 3` ✓

***

### 复杂度

* **时间**：$O(n \sqrt{n})$，外层枚举 $n$ 个目标值，内层枚举 $\sqrt{n}$ 个平方数

* **空间**：$O(n)$，一维 dp 数组

***

## C++ 代码

```c++
#include <vector>
#include <climits>
#include <algorithm>
using namespace std;

class Solution {
public:
    int numSquares(int n) {
        // dp[i] 表示凑成 i 所需完全平方数的最少个数
        vector<int> dp(n + 1, INT_MAX);
        dp[0] = 0;  // 凑成 0 不需要任何数

        for (int i = 1; i <= n; i++) {
            // 枚举所有不超过 i 的完全平方数 j*j
            for (int j = 1; j * j <= i; j++) {
                // 确保 dp[i - j*j] 是可达状态，再更新
                if (dp[i - j * j] != INT_MAX) {
                    dp[i] = min(dp[i], dp[i - j * j] + 1);
                }
            }
        }

        return dp[n];
    }
};
```

***

## 为什么不用贪心？

直觉上可能想：每次尽量选最大的平方数。

反例：`n = 12`

* 贪心：选 9，剩 3 → 选 1,1,1，共 **4个**

* 最优：选 4,4,4，共 **3个** ✓

贪心会错，必须 DP 枚举所有可能。
